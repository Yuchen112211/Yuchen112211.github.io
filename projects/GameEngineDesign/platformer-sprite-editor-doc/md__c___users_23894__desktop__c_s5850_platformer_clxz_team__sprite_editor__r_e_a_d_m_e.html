Sprite Editor

How to compile/install

  • First get into the directory under platformer-clxz-team/Game/SpriteEditor
  • Then run the command: python build_sprite_editor.py
  • Go into the bin folder, you will see a SpriteEditor.exe or SpriteEditor to execute the program

Example useage

  • You can enter the first text input with a bmp/png or whatever type of image as the spritesheet's file path, then the second blank as width of per frame, then third blank as height, forth as frame count of the animation.

    • For example, you can put in:

      1. ../asset/sprite.bmp
      2. 75
      3. 87
      4. 27

      Then press the button.

  • You can also only enter the first blank, for a specific format of our config file, to read the previously created sprite.

    • For example, you can put in :

      1. ../tmp/111.sed

      Then press the load button.

  • To save a sprite config file, you must load either from bmp file or from .sed file first
    • .sed is the suffix of the specified config file format.
    • You can try different dir, saving file without existent dir would not be saved.
