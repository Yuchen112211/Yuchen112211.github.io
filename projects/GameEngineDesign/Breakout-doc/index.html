Introduction

This is a famous arcade game since 1976, Nolan Bushnell did a really good job designing the game.

This version of the Breakout is implemented using SDL2 and C++, with OOD principle the code and class should be well organized.

Installation

Step 1: Download the code from git

git repo: https://github.com/Spring21BuildingGameEngines/monorepo-Yuchen112211/tree/main/Assignment1_Breakout

Go to your favorite location (file system, not the best coffee shop or steak house in town) and run

git clone xxx (I will probably update this as this should be another repo)

The problem is this game is developed as a course project of Northeastern University's CS5850, so the repo might not be public yet. Anyone interested can reach out to the author and I'll see what I can do

Step 2: Build the project

Now you have the code, impressive! If you are using windows, then I'm 100% sure that this would work. Although not that sure if your OS is MacOS or Linux

Build the project by run python script

python build.py

After the build, you should have a breakout.exe(or breakout if MacOS and Linux) lying in the bin folder

Step 3: Let's play!

Go into the bin folder, double click the breakout and start playing!
