The function basically handles how ball and paddle update, also detects the collision. It handles paddle movement, ball movement, collisions between ball, paddle and bricks. Also change the state of the game, which is changing variables in ResourceManager.

Crucial to determine whether the game hasn't end.

After update, check again for the game state.

Check the collision between ball and bricks.

Check if the ball has fallen to the bottom of the screen without colliding with the paddle. If so, reduce the lives and change the game state.
