Welcome to my first game project's web page!


The game is called "Breakout", which is a really classic game since 1976. My implementation of the game used SDL2 and C++ as basic tech stacks, with acute and precise collision detection, along with hand-picked background music and well-designed levels of games, this game would be fun to play.

Purpose of this site is to demonstrate the work that I have done, also show how to play the game with demo and preview.

In documentation section, there would be comments and documentations generated by doxygen, every detail about the code can be found in there.

In post mortem section, there are some plans about furthur development about the game if given more time and resource.

In preview section, I put in a demo video(6 min) and a preview video(1 min) in order to give you an idea how the game looks and how it's played.

In screenshot section you can see some screenshots of the game, there might be some scenarios in the pictures as they are hard to demonstrate in video, like clear out the whole level or choose another language.

Anyway, this is my first game and I am really excited about this! Hope you enjoy the site!
